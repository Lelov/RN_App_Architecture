## 依赖包
### 图标库：
[react-native-vector-icons](https://github.com/oblador/react-native-vector-icons)
查阅[全部图标](https://oblador.github.io/react-native-vector-icons/)，选中的图标库直接引入即可
```
import Icon from 'react-native-vector-icons/AntDesign';

...JS
/**
  * Icon 名字随意取，设置icon属性
  * size: 图标大小
  * name：需要显示的图标
  * color：图标颜色
  */

<Icon />
...
```
